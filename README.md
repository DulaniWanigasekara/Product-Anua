# ANUA – Peach 70% Niacin Serum Product Page

A responsive e-commerce landing page for the **ANUA Peach 70% Niacin Serum**, built with HTML5 and CSS3.

## 🛍️ Overview

This project is a single-page product showcase for the ANUA Peach 70% Niacin Serum. It features a clean dark-themed design with smooth hover animations, a fixed navigation bar, a product detail section with pricing, and call-to-action buttons.

## ✨ Features

- **Responsive layout** – adapts to desktop, tablet, and mobile screens (breakpoint at 995px)
- **Fixed navigation header** with logo and menu links
- **Product showcase** – circular image with red glow effect, product name, description, benefits list, and price
- **Social media links** – Twitter & Instagram icon buttons with hover effects
- **Add To Cart / Buy Now** action buttons with hover transitions
- **Smooth animations** – scale transforms and colour transitions on all interactive elements

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- *(Optional)* [VS Code](https://code.visualstudio.com/) with the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension for live reload during development

### Running locally

1. Clone the repository:

   ```bash
   git clone https://github.com/DulaniWanigasekara/Product-Anua.git
   ```

2. Open the project folder in VS Code (or any editor).

3. **Option A – Live Server (recommended):**  
   Right-click `index.html` → *Open with Live Server*.  
   The page will be served at `http://localhost:5502`.

4. **Option B – Direct file open:**  
   Double-click `index.html` to open it directly in your browser.
